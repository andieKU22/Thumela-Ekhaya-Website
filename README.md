# Thumela-Ekhaya-Website
Official single-page website for Thumela eKhaya courier and logistics, a Durban-based courier and logistics company specializing in reliable, affordable deliveries across cities, townships, and rural areas of South Africa. 
Live Website


Features
- Mordern, minimal, and mobile-responsive design
- Orange-nlack-white color scheme which shows energy, professionalim and displays trust
- Online delivery booking/request form which is powered by EmailJS
- Floating Whatsapp chat button
- Gallery showcasing delivery operations
- Testimonials and team introduction sections
- Sticky header navigation
- SEO-ready with meta tags and open graph support

Tchnologies used

- HTML5
- CSS3
- JavaScript
- EmailJS for form submissions 
- Google Fonts
- Font Awesome
